# Senior Smile Project

> AI 기반 시니어 정서 케어 스마트 액자 시스템

---

## 📌 프로젝트 소개
노인을 위한 정서 케어 AI 액자 시스템으로  
사진/영상 콘텐츠와 AI 비전을 활용하여 사용자 반응을 분석하고  
보호자에게 리포트를 제공하는 서비스입니다.

---

## 🎯 주요 기능
- 사진/영상 자동 재생
- AI 비전 기반 반응 분석 (미소, 시선)
- LLM 기반 리포트 생성
- 간단한 음성 대화

---

## 🧠 시스템 구성
- Raspberry Pi (Edge)
- MediaPipe (AI Vision)
- GPT / Gemini API (LLM)
