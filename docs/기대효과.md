# 기대 효과

## 1. 정서 안정

- 가족 콘텐츠를 통한 감정 자극
- 외로움 감소 및 심리적 안정 제공

---

## 2. 가족 간 소통 강화

- 보호자가 콘텐츠 제공
- 시스템이 반응을 분석하여 전달

→ 양방향 소통 구조 형성

---

## 3. 간접 케어 가능

- 보호자가 직접 방문하지 않아도 상태 확인 가능
- 돌봄 부담 감소

---

## 4. 데이터 기반 정서 관리

- 미소 및 시선 데이터를 기반으로 관심도 분석
- 사용자 상태를 간접적으로 파악 가능

---

## 5. 사회적 문제 완화

- 정서적 고립 감소
- 고독사 예방 보조 역할

---
