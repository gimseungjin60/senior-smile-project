# 유사 시스템 조사

## 1. 효돌 (AI 돌봄 인형)

### 특징
- 음성 기반 대화 기능 제공
- 약 복용 알림 및 일상 관리 기능
- 정서 케어 중심 서비스

### 한계
- 사용자 표정이나 반응을 분석하는 기능이 없음
- 데이터 기반 피드백 제공이 부족함

---

## 2. AI 스피커

### 특징
- 음성 명령을 통한 정보 제공 (날씨, 시간 등)
- 음악 재생 및 간단한 대화 가능

### 한계
- 시니어 대상 특화 기능 부족
- 감정 분석 및 보호자 연동 기능 미흡

---

## 3. 홈캠 (CCTV)

### 특징
- 실시간 영상 확인 가능
- 원격 모니터링 기능 제공

### 한계
- 정서 케어 기능 없음
- 단순 감시 기능에 집중

---

## 결론

기존 시스템들은 각각 기능은 존재하지만,  
사용자의 반응을 기반으로 분석하고 이를 보호자에게 전달하는 기능은 부족하다.

본 프로젝트는 이러한 한계를 보완하여  
정서 케어 중심의 새로운 접근을 제안한다.
